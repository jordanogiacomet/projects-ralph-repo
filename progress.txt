## Codebase Patterns
- Project lives in `/workspace/bianca-portfolio/` - all paths relative to that
- Use `next/font/google` for Playfair Display and DM Sans, exposed as CSS variables `--font-playfair` and `--font-dm-sans`
- Tailwind config maps these to `font-display` and `font-body` utility classes
- Custom color tokens defined both as CSS variables in globals.css and in tailwind.config.ts (e.g. `bg-primary`, `text-primary`, `accent`)
- Static export configured: `output: "export"` in next.config.mjs, images unoptimized
- Path alias `@/*` maps to project root
- Data files in `data/` directory, imported as JSON
- favicon.ico must be in `public/`, NOT `app/` — app/ favicon breaks static export builds
- All commands must be run from `/workspace/bianca-portfolio/`
- Build command: `npx next build`, Lint: `npx next lint`
- Animation pattern: Framer Motion `whileInView` with `viewport={{ once: true }}` for scroll-triggered entrances; `useScroll`+`useTransform` for parallax
- Framer Motion ease arrays need `as const` (e.g. `ease: [0.25, 0.1, 0.25, 1.0] as const`) for TypeScript compatibility
- Conditional variants with `useReducedMotion`: use `undefined` not `{}` to satisfy `Variants` type
- All animations use `useReducedMotion` hook to respect `prefers-reduced-motion`
