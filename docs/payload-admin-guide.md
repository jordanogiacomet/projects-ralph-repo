# Guia rápido do Payload Admin (Editores)

Este guia cobre operações comuns para equipes de conteúdo.

## 1) Criar um post (News)

1. Acesse `/admin` e faça login.
2. Abra `Posts` no menu lateral.
3. Clique em `Create New`.
4. Preencha no mínimo:
   - `title`
   - `slug` (ou deixe autoformatar)
   - `excerpt`
   - `content`
   - `status` = `published`
5. Opcionalmente preencha:
   - `featuredImage`
   - `meta.title`, `meta.description`, `meta.image`
   - `publishedAt`
6. Clique em `Save`.

## 2) Editar uma página institucional

1. Em `/admin`, abra `Pages`.
2. Localize a página desejada (`home`, `sobre`, `contato`, etc.).
3. Ajuste os campos de hero:
   - `heroType`
   - `heroTitle`
   - `heroSubtitle`
   - `heroImage`
4. Edite os blocos em `layout` conforme necessário.
5. Revise o grupo `meta` para SEO.
6. Clique em `Save` e valide no frontend.

## 3) Adicionar um cliente

1. Abra `Clientes`.
2. Clique em `Create New`.
3. Preencha:
   - `name`
   - `logo`
   - `segmento`
   - `order` (opcional para ordenação)
4. Clique em `Save`.

## 4) Ajustar Header/Footer/Configurações globais

1. No menu lateral, abra `Globals`.
2. Edite:
   - `Header` (menu, CTA, social links)
   - `Footer` (unidades, links sociais, copyright)
   - `Site Settings` (WhatsApp, cookie text, SEO default)
3. Salve e confirme no site.

## Boas práticas editoriais

- Use títulos objetivos e slugs em português, com hífens.
- Sempre preencha `meta.title` e `meta.description` para páginas/posts importantes.
- Em uploads, mantenha `alt` descritivo para acessibilidade e SEO.
